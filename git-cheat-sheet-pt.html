Useful Git commands


HEAD

git checkout {sha1} navega para a situação do sha1 (commit) específicado
git checkout HEAD navega para a situação inicial do HEAD
git revert {sha1} faz um novo commit desfazendo um commit específico
git revert HEAD faz um novo commit desfazendo o último commit
git reset {sha1} direciona o HEAD para o sha1, descartando os commits anteriores
git reset --merge {sha1} direciona o HEAD para o sha1, mantendo as alterações do working directory
git reset HEAD~1 --soft desfaz o último commit, voltando as alterações para o índice
git reset HEAD~1 desfaz o último commit, voltando as alterações para o working directory
git reset HEAD~1 --hard desfaz o último commit, descartando todas as alterações
git reset HEAD --hard desfaz todas as alterações do índice e do working directory
git bisect start inicia a busca de um commit com problema
git bisect bad HEAD marca o HEAD como defeituoso
git bisect good {sha1} marca um commit (último) bom
git bisect {good|bad} marca um commit intermediário
git remote [add|rm] {alias} {url} mapeia|remove um repositório remoto
git remote [-v] lista os repositórios remotos
git push envia os novos commits para um repositório
git rebase {branch} adiciona no branch atual o conteúdo do branch especificado
git rebase -i HEAD~3 junta os três últimos commits
git clone {url} baixa o conteúdo de um repositório
git pull {alias} [master] baixa as atualizações de um repositório e faz merge com a master
git pull [-s strategy] [-X options] [alias] [branch] baixa as atualizações utilizando estratégia e opções diferentes
git fetch baixa as atualizações de um repositório ou mais repositório (.git/FETCH_HEAD)
git merge {branch} faz merge do branch atual com o branch especificado
git merge {branch} --squash faz merge colocando todas as alterações no índice para um novo commit
git branch lista de branches locais
git branch {novo_branch} cria um novo branch
git branch -d {branch} excluí um branch
git checkout -b {novo_branch} cria e muda para um novo branch
git checkout {branch} muda para um branch existente
git tag v1.0 cria uma nova tag
git push origin v1.0 envia tag para o repositório remoto
git format-patch master --stdout > bugfix.patch faz um patch do branch atual contra o master
git apply --stat bugfix.patch lista as alterações do patch (sem efetivar modificações)
git am --signoff --ignore-whitespace ‹ bugfix.patch aplica o patch no branch corrente

INDEX

git commit -m 'comentário' faz um commit com as alterações adicionadas ao índice
git commit -am 'comentário' faz um commit adicionando os arquivos já controlados
git commit --amend ['comentário'] adiciona alterações no último commit
git reset HEAD desfaz as alterações (add) e volta os arquivos para o working directory
git reset {arquivo} desfaz as alterações (add) e volta o arquivo para o working directory
git reset {branch} {arquivo} desfaz as alterações (add) e volta o arquivo conforme o branch
git stash {save|pop|apply|drop|list} [stash] salva/recupera alterações do índice em uma área temporária
git stash -U [stash] salva alterações levando arquivos untrackeds

WORKING DIRECTORY

git add . adiciona arquivos alterados/novos ao índice (para o próximo commit)
git add {arquivo ou pasta} adiciona arquivo/pasta específica ao índice (para o próximo commit)
git rm {arquivo ou pasta} adiciona arquivo/pasta excluída ao índice (para o próximo commit)
git add -A . adiciona todas as alterações e exclusões ao índice
git clean -d -f apaga as alterações do working directory
git checkout {arquivo} volta o conteúdo do arquivo conforme o HEAD
git log [--since="1 weeks ago" ] exibe os commits
git log [-p] {arquivo} exibe as alterações de um determinado arquivo
git log {branch..master} exibe os commits que a branch não tem em relação a master
git log -S 'texto' exibe os commits que que afetaram o conteúdo contendo o texto
git log {diretório} exibe os commits que que afetaram o diretório
git status exibe as alterações do working directory e do índice
git reflog similar ao git log, porém exibe os commits apagados (reset)
gitk --all [arquivo] similar ao git log, porém exibe de forma gráfica
